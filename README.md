# Netflix Movies and TV Shows EDA Project

## Project Overview

This project analyzes the Netflix Movies and TV Shows dataset using Python.  
The goal is to clean the data, perform exploratory data analysis, create visualizations, and generate useful business insights from Netflix content trends.

This project was completed as part of the AI & Machine Learning Internship Project 01: Exploratory Data Analysis & Insights Report.

---

## Dataset

Dataset used: Netflix Movies & TV Shows Dataset  
Source: Kaggle  
Dataset link: https://www.kaggle.com/datasets/shivamb/netflix-shows

The dataset contains information about Netflix titles including:

- Title
- Type: Movie or TV Show
- Director
- Cast
- Country
- Date Added
- Release Year
- Rating
- Duration
- Genre
- Description

## Project Workflow

1. Loaded and inspected the dataset
2. Checked shape, data types, missing values, and duplicates
3. Cleaned missing values and fixed date columns

## EDA Questions Answered

1. How many Movies and TV Shows are available on Netflix?
2. Which countries produce the most Netflix content?
3. In which years did Netflix add the most content?
4. What are the most common Netflix genres?
5. What are the most common content ratings on Netflix?

---

## Key Insights

1. Netflix has more Movies than TV Shows. The dataset contains 6,131 Movies and 2,578 TV Shows.
2. Netflix added the highest number of titles in 2019, with 1,999 titles.
3. International Movies are the most common genre, with 2,752 titles.
4. Dramas and Comedies are also highly common categories on Netflix.
5. TV-MA is the most common rating, showing that much of Netflix's content is targeted toward mature audiences.

---

## Most Surprising Finding

The most surprising finding was that International Movies are the most common genre on Netflix.  
This shows that Netflix strongly focuses on global content and international audiences.

---

## Conclusion

This project helped in understanding Netflix's content strategy through data analysis.  
The analysis shows that Netflix has a strong focus on movies, international content, drama, comedy, and mature audience categories.
